      KiCad Library Manager watches a folder for downloaded KiCad components and helps you import them into organized library repositories — locally or via Git.

      First-Time Setup

      1. Click Browse and select your KiCad root library folder (e.g. C:\Users\you\Documents\KiCad\libs).
      2. Click Save & Continue.
      3. In Settings → Libraries, choose a Watch Folder — this is where you drop downloaded parts (e.g. your Downloads folder).
      4. Add at least one repository and mark it as default (⭐).

      Importing a Component

      1. Download a KiCad component (e.g. from SnapEDA or Component Search Engine) into the watch folder — or drag & drop it onto the app window.
      2. The app detects it automatically and shows the filename with an analysis summary.
      3. Select the Target Library and Category (the app pre-fills a suggestion).
      4. Click OK to import. The component is copied into the correct repo folder and registered in KiCad's library tables.
      5. Use Skip to discard the current file, or Hide to dismiss the window while keeping queued files.

      Managing Libraries (Repositories)

      • Local Folder — creates a new subfolder inside your KiCad library root for organising parts locally.
      • Git Repository — clones a remote Git repo (e.g. a shared team library on GitHub). The URL is validated before cloning.
      • Click ⭐ to set a repository as the default import destination.
      • Click 🗑️ to remove a repository from the app. Files on disk are not deleted.

      Managing Categories

      • Categories define the sub-folder structure inside each repository (e.g. symbols/Passives.kicad_sym).
      • The app automatically suggests the best category when a component is detected, based on its description and keywords.
      • Go to Settings → Categories to add, rename, or delete categories.
      • Click ✏️ to rename inline — press Enter to save or Esc to cancel.
      • Click 🗑️ to delete a category. Physical library files are not renamed or deleted.
      • The last remaining category cannot be deleted.

      Git Sync

      • The ↻ button in the main view syncs all Git-backed libraries immediately.
      • The cloud icon (☁️ / ✅ / ⚠️) shows the current sync state at a glance.
      • ⚠️ means a repo is behind its remote or unreachable — check your internet connection or repo URL.

      Rollback / Undo

      • Settings → System & History shows a log of recent imports.
      • Click Rollback Last on the most recent entry to undo it. Older entries cannot be individually reverted.
      • Rollback removes the imported files and restores the previous .kicad_sym library table state.

      Autostart

      Enable Run automatically on system startup in System & History to have the app start silently at login and monitor your watch folder in the background. On macOS, this installs a LaunchAgent plist in ~/Library/LaunchAgents/. On Windows, it adds a Registry Run key.
